# Driver Safety Monitoring System

This project detects **driver drowsiness** and **possible crash risks** from video streams using computer vision.  
It provides a simple **Flask web interface** where users can upload a video and view real-time processed output.

---

## Features

- **Drowsiness Detection** using MediaPipe FaceMesh  
- **Crash/FCW Detection** using YOLOv8  
- **Web Interface** built with Flask
